# Interview Questions

1.  What makes a password strong? Length, complexity, and uniqueness.

2.  Common password attacks? Brute force, dictionary, credential
    stuffing.

3.  Why is password length important? It increases search space.

4.  What is a dictionary attack? Trying common words/passwords.

5.  What is MFA? Multiple authentication factors.

6.  Password managers? Generate/store unique passwords.

7.  Passphrases? Long memorable phrases.

8.  Common mistakes? Reuse, short passwords, personal info.
